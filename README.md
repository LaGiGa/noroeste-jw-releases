# Noroeste JW - Sistema de Gerenciamento de Designações

Este é um sistema completo para gerenciamento de discursos e designações para congregações das Testemunhas de Jeová.

## Funcionalidades

- **Dashboard**: Visão geral de publicadores, reuniões e discursos.
- **Irmãos**: Cadastro e gerenciamento de publicadores, com filtros e busca.
- **Discursos**: Lista de discursos públicos, temas, e histórico.
- **Designações**: Visualização e geração de designações para reuniões de meio de semana e fim de semana.
- **Escola do Ministério**: Acompanhamento de estudantes e designações.

## Tecnologias Utilizadas

- React 19
- TypeScript
- Tailwind CSS v4
- Vite
- Lucide React (Ícones)
- React Router Dom

## Como Rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse no navegador: `http://localhost:5173`

## Estrutura do Projeto

- `src/components`: Componentes reutilizáveis e Layout.
- `src/pages`: Páginas principais da aplicação.
- `src/types`: Definições de tipos TypeScript.
- `src/data`: Dados mockados para teste.
